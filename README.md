# laravel10-vue-inertia
Laravel10 vue3 inertia vite

Full manual: https://nimb.ws/lyanwE

## 1-Install laravel valet composer
```
composer global require laravel/valet
```
## 2
```
valet install
```
## 3
```
valet link
```
```
valet secure
```

## 4
```
cp .env.example .env
```
```
php artisan key:generate
```

## 5
```
composer install
```
```
npm i && npm run dev
```
